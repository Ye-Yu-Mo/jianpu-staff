# 简谱文本格式规范

本项目使用一套纯文本简谱格式，设计目标是易于键盘输入、无歧义、可机器解析。

---

## 音高

用数字 `1`~`7` 表示 do~ti，`0` 表示休止符。

```
1 2 3 4 5 6 7 0
```

默认调号为 C 大调，默认八度为第 4 组（中央 C 所在组）。

---

## 八度

在音符后附加符号改变八度：

| 符号 | 含义         | 示例   | 音名 |
|------|--------------|--------|------|
| `'`  | 升高一个八度 | `1'`   | C5   |
| `''` | 升高两个八度 | `1''`  | C6   |
| `,`  | 降低一个八度 | `1,`   | C3   |
| `,,` | 降低两个八度 | `1,,`  | C2   |

---

## 时值

在音符后附加符号改变时值，默认为四分音符：

| 符号  | 时值           | 拍数   | 示例   |
|-------|----------------|--------|--------|
| 无    | 四分音符       | 1 拍   | `1`    |
| `-`   | 二分音符       | 2 拍   | `1-`   |
| `--`  | 全音符         | 4 拍   | `1--`  |
| `/`   | 八分音符       | ½ 拍   | `1/`   |
| `//`  | 十六分音符     | ¼ 拍   | `1//`  |
| `*`   | 附点四分音符   | 1½ 拍  | `1*`   |
| `*/`  | 附点八分音符   | ¾ 拍   | `1*/`  |

八度符号和时值符号可以组合，八度在前，时值在后：

```
1'-    → C5，二分音符
1,/    → C3，八分音符
1''*   → C6，附点四分音符
1'*/   → C5，附点八分音符
```

---

## 小节线

用 `|` 分隔小节，末尾的 `|` 可省略：

```
1 2 3 4 | 5 6 7 1' |
```

每个小节的音符时值之和必须等于拍号，否则报错。

---

## 头部指令

写在最前面，用方括号包裹，顺序不限：

### 拍号

```
[4/4]   四四拍（默认）
[3/4]   四三拍
[2/4]   四二拍
[6/8]   八六拍
```

### 调号

```
[1=C]   C 大调（默认）
[1=G]   G 大调（1 个升号）
[1=D]   D 大调（2 个升号）
[1=A]   A 大调（3 个升号）
[1=E]   E 大调（4 个升号）
[1=B]   B 大调（5 个升号）
[1=F]   F 大调（1 个降号）
```

调号决定数字到音名的映射。例如 `[1=G]` 时，`1`=G、`2`=A、`3`=B、`4`=C（升八度）、`5`=D、`6`=E、`7`=F#。

---

## 行内指令

可出现在乐谱任意位置（小节内或小节线前后），用方括号包裹。

### 速度

```
[q=120]     四分音符 = 120 BPM
[q=60]      四分音符 = 60 BPM
[Allegro]   速度文字标记（不影响 MIDI 播放，仅显示）
[Andante]
[Moderato]
```

### 力度

| 指令    | 含义         |
|---------|--------------|
| `[ppp]` | 极弱         |
| `[pp]`  | 很弱         |
| `[p]`   | 弱           |
| `[mp]`  | 中弱         |
| `[mf]`  | 中强         |
| `[f]`   | 强           |
| `[ff]`  | 很强         |
| `[fff]` | 极强         |
| `[<]`   | 渐强（cresc.）|
| `[>]`   | 渐弱（dim.） |

行内力度示例：

```
[4/4] [1=C] [mf] 1 2 3 4 | [f] 5 6 7 1' | [p] 1'-- |
```

---

## 延音线

用 `~` 后缀标记延音线起点，下一个相同音高的音符为终点。延音线只能连接相同音高。

```
1~ 1        → C4 延音线到下一个 C4（合并为一个持续音）
1-~ | 1-    → C4 二分音符延音线跨小节到下一个 C4 二分音符
```

规则：
- `~` 紧跟在时值符号之后（如 `1-~`、`1/~`）
- 延音线终点音符不加 `~`
- 延音线不能跨越不同音高

---

## 连音组（三连音 / 其他连音）

用花括号 `{}` 包裹连音组内的音符，默认为三连音。非三连音需在冒号前写明音符数：

```
{1/ 2/ 3/}          → 三连音（3 个八分音符占 1 拍）
{5: 1/ 2/ 3/ 4/ 5/} → 五连音（5 个八分音符占 2 拍）
{2: 1 2}            → 二连音（2 个四分音符占 3 拍，用于 3/4 拍）
```

规则：
- 花括号内的音符时值必须一致
- 三连音：3 个音符占 2 个同类音符的时值
- n 连音：n 个音符占最近的 2 的幂次个同类音符的时值
- 连音组内可使用八度标记，不可嵌套

---

## 装饰音

装饰音写在主音符之前，用小写字母前缀标记：

| 语法       | 含义                   | MusicXML 对应         |
|------------|------------------------|-----------------------|
| `g1`       | 前倚音（grace note）   | `<grace/>`            |
| `g1 2`     | 前倚音 g1 装饰主音 2   | grace note + 主音符   |
| `1tr`      | 颤音（trill）          | `<trill-mark/>`       |
| `1mor`     | 波音（mordent）        | `<mordent/>`          |
| `1turn`    | 回音（turn）           | `<turn/>`             |

规则：
- 前倚音 `g` 前缀只能修饰紧随其后的单个音符，不占拍
- `tr`、`mor`、`turn` 是音符后缀，紧跟在时值符号之后（如 `1-tr`）
- 装饰音后缀与八度符号的顺序：`1'tr`（八度在前，装饰音在后）

---

## 技巧标记

技巧标记是音符后缀，紧跟在时值符号（和装饰音后缀）之后：

| 后缀  | 含义                   | MusicXML 对应              |
|-------|------------------------|----------------------------|
| `.`   | 断奏（staccato）       | `<staccato/>`              |
| `..`  | 极短断奏（staccatissimo）| `<staccatissimo/>`       |
| `^`   | 重音（accent）         | `<accent/>`                |
| `^^`  | 强重音（marcato）      | `<strong-accent/>`         |
| `_`   | 连奏（tenuto）         | `<tenuto/>`                |
| `^.`  | 重音 + 断奏            | `<accent/>` + `<staccato/>`|

组合顺序：`音高` + `八度` + `时值` + `装饰音` + `技巧`

```
1'.^    → C5，四分音符，重音 + 断奏
1-_     → C4，二分音符，连奏
1tr.    → C4，四分音符，颤音，断奏
```

---

## 符号组合顺序（完整规则）

一个完整音符 token 的结构：

```
[g] digit [八度] [时值] [装饰音后缀] [技巧后缀] [~]
```

示例：

```
g1 2'.^~    → 前倚音 C4，主音 C5 四分音符，重音+断奏，延音线起点
```

---

## 完整示例

```
[4/4] [1=G] [q=96] [mf] 1 2 3- | [f] 4 3 2 1 | {1/ 2/ 3/} 4 5- | 1'--~ | 1'-- |
[p] g7 1'. 2/ 3/ 4 | 1tr 2 3 4 | [<] 5 6 7 1' | [ff] 1'-- |
```

逐小节解析：

| 小节 | 内容                     | 说明                                                  |
|------|--------------------------|-------------------------------------------------------|
| 1    | `[mf] 1 2 3-`            | 中强，G4 四分、A4 四分、B4 二分                       |
| 2    | `[f] 4 3 2 1`            | 强，C5 B4 A4 G4 各四分                                |
| 3    | `{1/ 2/ 3/} 4 5-`        | 三连音（G4 A4 B4）+ C5 四分 + D5 二分                 |
| 4    | `1'--~`                  | G5 全音符，延音线起点                                 |
| 5    | `1'--`                   | G5 全音符，延音线终点                                 |
| 6    | `[p] g7 1'. 2/ 3/ 4`     | 弱，前倚音 F#5 装饰 G5（断奏）+ A5 八分 + B5 八分 + C5 四分 |
| 7    | `1tr 2 3 4`              | G4 颤音 + A4 B4 C5 各四分                             |
| 8    | `[<] 5 6 7 1'`           | 渐强，D5 E5 F#5 G5 各四分                             |
| 9    | `[ff] 1'--`              | 极强，G5 全音符                                       |

---

## 错误处理

| 错误类型           | 示例输入              | 错误信息                                  |
|--------------------|-----------------------|-------------------------------------------|
| 未知符号           | `1 x 3`               | 未知符号 'x'                              |
| 小节时值不足       | `[4/4] 1 2 3`         | 第1小节时值不足：期望4拍，实际3拍         |
| 小节时值超出       | `[4/4] 1-- 2 3`       | 第1小节时值超出：期望4拍，实际6拍         |
| 延音线音高不匹配   | `1~ 2`                | 延音线终点音高不匹配：起点 C4，终点 D4    |
| 连音组音符数不符   | `{1/ 2/}`             | 三连音需要3个音符，实际2个                |
| 连音组时值不一致   | `{1/ 2 3/}`           | 连音组内音符时值必须一致                  |
| 前倚音无主音符     | `g1`（行末）          | 前倚音后缺少主音符                        |
